
# Qusicians Backend

A Node.js backend for **Qusicians**, a collaborative Spotify jukebox app.  
This server handles Spotify OAuth onboarding, session management, and API endpoints for interacting with Spotify.

## Qusicians 

Qusicians is a collaborative Spotify jukebox app. This Node.js backend handles Spotify OAuth onboarding, session management, and provides API endpoints for interacting with Spotify, such as retrieving songs, viewing the queue, and adding tracks to a session. It is designed with a modular and scalable structure to support multiple users and sessions in parallel.

All source code lives in the `src/` folder. The `config/` folder contains public constants like Spotify URLs, OAuth scopes, and callback paths. The `controllers/` folder handles HTTP request and response logic, while `services/` encapsulate business logic and Spotify API interactions. The `models/` folder defines session and user data structures, and `helpers/` contains utility functions. API endpoints are defined in the `routes/` folder and mapped to controllers. The server entry point is `src/server.js`, and sensitive environment variables, such as Spotify client ID and secret, are stored in a `.env` file, which is gitignored.

## Getting Started

1. Clone the repository and install dependencies:
```bash
git clone <repo-url>
cd qusicians-backend
npm install
```
2. Create a .env file in the root with the following contents:

SPOTIFY_CLIENT_ID=your_client_id
SPOTIFY_CLIENT_SECRET=your_client_secret
SPOTIFY_REDIRECT_URI=http://127.0.0.1:3000/callback
PORT=3000

3. Start the local server:
```bash
node src/server.js
```

4. Visit http://127.0.0.1:3000/login to start Spotify OAuth login.
- Authorise with your Spotify credentials (they must have been added as dev user in Spotify developer console)

## NOTES:
- .env is gitignored to protect your Spotify credentials.
- Spotify URLs, OAuth scopes, and other constants are stored in src/config/constants.js.
- This is a development-ready backend; sessions and persistent storage can be added later.
